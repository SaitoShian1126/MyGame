◆動作確認済み環境◆

（DirectxInput対応のゲームパッドに対応しています。）

◆操作方法◆
//キーボード
W : 上移動
A : 左移動
S : 下移動
D : 右移動
C : 弾切り替え
SPACE(長押し) : 弾発射

//ゲームパッド
十字の上 : 上移動
十字の左 : 左移動
十字の下 : 下移動
十字の右 : 右移動
B : 弾切り替え
右ボタン上 : 弾発射
